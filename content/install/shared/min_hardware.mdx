## Minimum Hardware Requirements


| Component | Specification |
| --- | --- |
| CPU | Two socket based server with at least 8 cores of one of these CPU types: <br/> * *ARM64* / *Axion* / *Gravitron* <br/> * *Intel (or compatible)* x86-64 with AVX2 <br/> * *Power PC* 8le |
| GPU | See [GPU Driver](/content/install/shared/gpu_driver) below for the list of supported GPUs |
| Memory | Minimum 8GB |
| Hard Drive | SSD or SATA 7200RPM hard drive with 4X memory capacity |
