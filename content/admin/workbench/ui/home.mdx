*Home* serves as the landing page for
[Workbench](/content/admin/workbench).


<Frame caption="Workbench Home Page">
  <img src="/content/admin/workbench/images/wb.home.png" alt="../../images/wb.home.png" />
</Frame>

This landing page can be reached at any time by clicking on the
**Home** menu item at the top.  This will open several options in
the right-hand pane, including:

* **Getting Started** - links to various tutorials
  native API reference, and native API downloads
* **Import** - links to the import & external table
  wizards, as well as to the various data source specific import wizards
* **Connect** - sample code blocks for connecting to
  *Kinetica* with the various native API languages and JDBC
* **Resources** - links to video tutorials
