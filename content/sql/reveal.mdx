## LOAD DASHBOARD


*Kinetica* provides a SQL interface for loading
[Reveal](/content/bi/reveal) *dashboards* from a variety of sources:

* [KiFS](/content/tools/kifs) path
* Cloud provider via [data source](/content/sql/ddl#sql-create-data-source)
* Public URL (GitHub, etc.)


```sql title="LOAD DASHBOARD Syntax"
LOAD DASHBOARD "<dashboard name>"
FROM FILE PATH '<file path>'
[WITH OPTIONS (DATASOURCE = '<data source name>')]
```

| Parameters | Description |
| --- | --- |
| `<dashboard name>` | Name of the *dashboard*. <br/> <Note> This name **must** match the name of the *dashboard* in the file. </Note> |
| `<file path>` | *Dashboard* source file path. <br/> If a *data source* is specified, the path must resolve to an accessible file at that location. <br/> If no *data source* is specified, the file path can refer to one of two locations: <br/> * Public URL <br/> * [KiFS](/content/tools/kifs) path <br/> <br/> `kifs://reveal/db/nyctaxi.db`: File <Badge color="gray">nyctaxi.db</Badge> , under the *KiFS* virtual directory <Badge color="gray">reveal/db</Badge> |
| `<data source name>` | Name of *data source* through which the *dashboard* will be loaded. |


For example, to load a *dashboard* named `nyctaxi.db` into *Reveal*:

<CodeGroup>

{/* Source: /examples/sql/reveal.kisql (lines 40-41) */}
```sql KiFS
LOAD DASHBOARD "NYC Taxi"
FROM FILE PATH 'kifs://reveal/nyctaxi.db'
```

{/* Source: /examples/sql/reveal.kisql (lines 45-47) */}
```sql Data Source
LOAD DASHBOARD "NYC Taxi"
FROM FILE PATH 'nyctaxi.db'
WITH OPTIONS (DATASOURCE = 'reveal')
```

{/* Source: /examples/sql/reveal.kisql (lines 51-52) */}
```sql Public URL
LOAD DASHBOARD "NYC Taxi"
FROM FILE PATH 'https://github.com/kineticadb/examples/raw/master/quickstart/nyctaxi.db'
```

</CodeGroup>
