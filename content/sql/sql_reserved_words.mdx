---
title: SQL Reserved Word List
---

*Kinetica* reserves the following words within its SQL context.  Any tables,
columns, or other named references that conflict with these must be
double-quoted to be used.

For instance, if using a column name of *abs*, it must be double-quoted:

{/* Source: /examples/sql/reserved_word.kisql (lines 43-46) */}
```sql
CREATE TABLE example.quoted_reserved_word_column_name
(
    "abs" INT
)
```
