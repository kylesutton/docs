## Monitor Target


<CodeGroup>

{/* Source: /examples/sql/stream/target.kisql (lines 35-36) */}
```sql Data Sink
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS (DATASINK_NAME = 'kin_dsink')
```

{/* Source: /examples/sql/stream/target.kisql (lines 44-46 48-50) */}
```sql Kafka (No Auth)
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS
(
	DESTINATION = 'kafka://kafka-test:9092',
	KAFKA_TOPIC_NAME = 'kafka-test'
)
```

{/* Source: /examples/sql/stream/target.kisql (lines 58-59) */}
```sql Local Table
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS (DESTINATION = 'table://example.order_target')
```

{/* Source: /examples/sql/stream/target.kisql (lines 67-68) */}
```sql HTTP
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS (DESTINATION = 'http://localhost:8080')
```

{/* Source: /examples/sql/stream/target.kisql (lines 76-77) */}
```sql HTTPS
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS (DESTINATION = 'https://localhost:8088')
```

</CodeGroup>


## Refresh


<CodeGroup>

{/* Source: /examples/sql/stream/refresh.kisql (lines 32-33) */}
```sql On Change (default)
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS (DATASINK_NAME = 'kin_dsink')
```

{/* Source: /examples/sql/stream/refresh.kisql (lines 41-43) */}
```sql On Change (explicit)
CREATE STREAM kin_stream ON example.order_stream
REFRESH ON CHANGE
WITH OPTIONS (DATASINK_NAME = 'kin_dsink')
```

{/* Source: /examples/sql/stream/refresh.kisql (lines 51-53) */}
```sql Scheduled
CREATE STREAM kin_stream ON example.order_stream
REFRESH EVERY 30 SECONDS
WITH OPTIONS (DATASINK_NAME = 'kin_dsink')
```

{/* Source: /examples/sql/stream/refresh.kisql (lines 61-63) */}
```sql Scheduled w/ Start Time
CREATE STREAM kin_stream ON example.order_stream
REFRESH EVERY 1 MINUTE STARTING AT '2025-02-25 02:25:00'
WITH OPTIONS (DATASINK_NAME = 'kin_dsink')
```

</CodeGroup>

## Options


<CodeGroup>

{/* Source: /examples/sql/stream/options.kisql (lines 32-34) */}
```sql Filter
CREATE STREAM kin_stream ON example.order_stream
WHERE cost > 10000
WITH OPTIONS (DATASINK_NAME = 'kin_dsink')
```

{/* Source: /examples/sql/stream/options.kisql (lines 42-47) */}
```sql Increasing Column
CREATE STREAM kin_stream ON example.order_stream
WITH OPTIONS
(
	DATASINK_NAME = 'kin_dsink',
	INCREASING_COLUMN = 'order_id'
)
```

</CodeGroup>
