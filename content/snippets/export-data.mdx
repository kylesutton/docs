## EXPORT ... INTO


### Sources (Table/Query)


<CodeGroup>

{/* Source: /examples/sql/export/source.kisql (lines 45-46) */}
```sql Table
EXPORT TABLE employee
INTO FILE PATH '/export/employee.csv'
```

{/* Source: /examples/sql/export/source.kisql (lines 52-58) */}
```sql Query
EXPORT QUERY
(
    SELECT id, manager_id, first_name, last_name, salary, hire_date
    FROM employee
    WHERE dept_id = 2
)
INTO FILE PATH '/export/employee_dept2.csv'
```

</CodeGroup>


### Data Sinks (File/Table/DML)


<CodeGroup>

{/* Source: /examples/sql/export/data_sink.kisql (lines 90-92) */}
```sql File
EXPORT TABLE employee
INTO FILE PATH '/data/employee.csv'
WITH OPTIONS (DATASINK_NAME = 's3_dsink')
```

{/* Source: /examples/sql/export/data_sink.kisql (lines 98-100) */}
```sql Table (via JDBC)
EXPORT TABLE employee
INTO REMOTE TABLE 'example.remote_employee'
WITH OPTIONS (DATASINK_NAME = 'jdbc_dsink')
```

{/* Source: /examples/sql/export/data_sink.kisql (lines 128-130) */}
```sql DML (via JDBC)
EXPORT TABLE employee
INTO REMOTE QUERY 'INSERT INTO example.remote_employee VALUES (?, ?, ?, ?, ?, ?, ?)'
WITH OPTIONS (DATASINK_NAME = 'jdbc_dsink')
```

{/* Source: /examples/sql/export/jdbc_init.kisql (lines 109-116) */}
```sql Init Options (via JDBC)
EXPORT TABLE ts_source
INTO REMOTE TABLE 'ts'
WITH OPTIONS
(
	DATASINK_NAME = 'jdbc_dsink',
	JDBC_SESSION_INIT_STATEMENT = 'SET TIMEZONE=''EST''',
	JDBC_CONNECTION_INIT_STATEMENT = 'CREATE TABLE ts (ts DATETIME)'
)
```

</CodeGroup>


### Delimited Text Options


<CodeGroup>

{/* Source: /examples/sql/export/formats.kisql (lines 45-46) */}
```sql CSV

```

```sql Forced Format

```

{/* Source: /examples/sql/export/formats.kisql (lines 45-47) */}

{/* Source: /examples/sql/export/formats.kisql (lines 61-63) */}
```sql Columns Mapped by Name

INTO FILE PATH '/export/products_name.csv'

```

{/* Source: /examples/sql/export/formats.kisql (lines 69-71) */}
```sql Columns Ignored by Position
EXPORT TABLE product
INTO FILE PATH '/export/products_pos.csv'
WITH OPTIONS (FIELDS IGNORED BY POSITION(2, 4))
```

{/* Source: /examples/sql/export/formats.kisql (lines 77-84) */}
```sql Parsing Options
EXPORT TABLE product
INTO FILE PATH '/export/products.ssv'
FORMAT TEXT
(
    DELIMITER = ';',
    INCLUDES HEADER = TRUE,
    NULL = '<null>'
)
```

{/* Source: /examples/sql/export/local_file_compression.kisql (lines 59-61) */}
```sql GZip Compression
EXPORT TABLE product
INTO FILE PATH '/export/products.csv.gz'
WITH OPTIONS (COMPRESSION_TYPE = 'gzip')
```

### Naming Options


<CodeGroup>

{/* Source: /examples/sql/export/local_file_naming.kisql (lines 87-91) */}
```sql Single File w/o Code
-- Creates a file with a name like:
--   export/product.csv
EXPORT TABLE product
INTO FILE PATH '/export/product.csv'
WITH OPTIONS (SINGLE_FILE = 'overwrite')
```

{/* Source: /examples/sql/export/local_file_naming.kisql (lines 49-52) */}
```sql Single File w/ Code
-- Creates a file with a name like:
--   export/product_611460000.csv
EXPORT TABLE product
INTO FILE PATH '/export/product.csv'
```

{/* Source: /examples/sql/export/local_file_naming.kisql (lines 107-112) */}
```sql Multiple Files w/ Code
-- Creates a set of files with names like:
-- * export/product_611641000.csv
-- * export/product_611642000.csv
EXPORT TABLE product
INTO FILE PATH '/export/product.csv'
WITH OPTIONS (SINGLE_FILE = false)
```

</CodeGroup>
