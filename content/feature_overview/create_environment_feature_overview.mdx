<a id="create-environment-feature-overview"></a>


Creates a new environment which can be used by [user-defined functions](../../concepts/udf/) (UDF).


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| environment_name | string | Name of the environment to be created. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| environment_name | string | Value of input parameter *environment_name*. |
| info | map of string to strings | Additional information. |
