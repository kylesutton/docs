<a id="drop-datasource-feature-overview"></a>


Drops an existing [data source](../../concepts/data_sources/).  Any external
tables that depend on the data source must be dropped before it can be dropped.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Name of the data source to be dropped. Must be an existing data source. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Value of input parameter *name*. |
| info | map of string to strings | Additional information. |
