<a id="delete-directory-feature-overview"></a>


Deletes a directory from [KiFS](../../tools/kifs/).


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| directory_name | string | Name of the directory in KiFS to be deleted. The directory must contain no files, unless *recursive* is *true* |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). <br/> <br/> recursive: If *true*, will delete directory and all files residing in it. If false, directory must be empty for deletion. The default value is `false`. The supported values are: * true * false <br/> no_error_if_not_exists: If *true*, no error is returned if specified directory does not exist The default value is `false`. The supported values are: * true * false |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| directory_name | string | Value of input parameter *directory_name*. |
| info | map of string to strings | Additional information. |
