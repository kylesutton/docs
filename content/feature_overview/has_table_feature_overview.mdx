<a id="has-table-feature-overview"></a>


Checks for the existence of a table with the given name.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| table_name | string | Name of the table to check for existence, in [schema_name.]table_name format, using standard [name resolution rules](../../concepts/tables/#table-name-resolution). |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| table_name | string | Value of input parameter *table_name* |
| table_exists | boolean | Indicates whether the table exists or not.  Possible values are: <br/> * true <br/> * false |
| info | map of string to strings | Additional information. |
