<a id="has-proc-feature-overview"></a>


Checks the existence of a proc with the given name.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| proc_name | string | Name of the proc to check for existence. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| proc_name | string | Value of input parameter *proc_name* |
| proc_exists | boolean | Indicates whether the proc exists or not.  Possible values are: <br/> * true <br/> * false |
| info | map of string to strings | Additional information. |
