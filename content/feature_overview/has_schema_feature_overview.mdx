<a id="has-schema-feature-overview"></a>


Checks for the existence of a schema with the given name.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| schema_name | string | Name of the schema to check for existence, in root, using standard [name resolution rules](../../concepts/tables/#table-name-resolution). |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| schema_name | string | Value of input parameter *schema_name* |
| schema_exists | boolean | Indicates whether the schema exists or not.  Possible values are: <br/> * true <br/> * false |
| info | map of string to strings | Additional information. |
