<a id="alter-schema-feature-overview"></a>


Used to change the name of a SQL-style [schema](../../concepts/schemas/), specified in input parameter *schema_name*.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| schema_name | string | Name of the schema to be altered. |
| action | string | Modification operation to be applied <br/> <br/> add_comment: Adds a comment describing the schema <br/> rename_schema: Renames a schema to input parameter *value*. Has the same naming restrictions as [tables](../../concepts/tables/). |
| value | string | The value of the modification, depending on input parameter *action*.  For now the only value of input parameter *action* is *rename_schema*.  In this case the value is the new name of the schema. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| schema_name | string | Value of input parameter *schema_name*. |
| info | map of string to strings | Additional information. |
