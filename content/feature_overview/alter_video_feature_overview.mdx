<a id="alter-video-feature-overview"></a>


Alters a video.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| path | string | Fully-qualified [KiFS](../../tools/kifs/) path to the video to be altered. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). <br/> <br/> ttl: Sets the [TTL](../../concepts/ttl/) of the video. |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| path | string | Fully qualified KIFS path to the video file. |
| info | map of string to strings | Additional information. |
