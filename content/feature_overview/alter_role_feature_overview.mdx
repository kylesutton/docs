<a id="alter-role-feature-overview"></a>


Alters a Role.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Name of the role to be altered. Must be an existing role. |
| action | string | Modification operation to be applied to the role. <br/> <br/> set_comment: Sets the comment for an internal role. <br/> set_resource_group: Sets the resource group for an internal role. The resource group must exist, otherwise, an empty string assigns the role to the default resource group. |
| value | string | The value of the modification, depending on input parameter *action*. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Value of input parameter *name*. |
| info | map of string to strings | Additional information. |
