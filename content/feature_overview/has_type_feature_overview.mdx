<a id="has-type-feature-overview"></a>


Check for the existence of a type.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| type_id | string | Id of the type returned in response to [Create Type](/content/feature_overview/create_type_feature_overview#create-type-feature-overview) request. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| type_id | string | Value of input parameter *type_id*. |
| type_exists | boolean | Indicates whether the type exists or not.  Possible values are: <br/> <br/> * true <br/> * false |
| info | map of string to strings | Additional information. |
