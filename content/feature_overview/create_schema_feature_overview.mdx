<a id="create-schema-feature-overview"></a>


Creates a SQL-style [schema](../../concepts/schemas/). Schemas are containers for tables and views.  Multiple tables and views can be defined with the same name in different schemas.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| schema_name | string | Name of the schema to be created.  Has the same naming restrictions as [tables](../../concepts/tables/). |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). <br/> <br/> no_error_if_exists: If *true*, prevents an error from occurring if the schema already exists. The default value is `false`. The supported values are: * true * false |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| schema_name | string | Value of input parameter *schema_name*. |
| info | map of string to strings | Additional information. |
