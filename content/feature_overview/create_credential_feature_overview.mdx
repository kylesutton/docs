<a id="create-credential-feature-overview"></a>


Create a new [credential](../../concepts/credentials/).


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| credential_name | string | Name of the credential to be created. Must contain only letters, digits, and underscores, and cannot begin with a digit. Must not match an existing credential name. |
| type | string | Type of the credential to be created.  The supported values are: <br/> <br/> * aws_access_key <br/> * aws_iam_role <br/> * azure_ad <br/> * azure_oauth <br/> * azure_sas <br/> * azure_storage_key <br/> * confluent <br/> * docker <br/> * gcs_service_account_id <br/> * gcs_service_account_keys <br/> * hdfs <br/> * jdbc <br/> * kafka <br/> * nvidia_api_key <br/> * openai_api_key <br/> * rest |
| identity | string | User of the credential to be created. |
| secret | string | Password of the credential to be created. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| credential_name | string | Value of input parameter *credential_name*. |
| info | map of string to strings | Additional information. |
