<a id="alter-credential-feature-overview"></a>


Alter the properties of an existing [credential](../../concepts/credentials/).


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| credential_name | string | Name of the credential to be altered. Must be an existing credential. |
| credential_updates_map | map of string to strings | Map containing the properties of the credential to be updated. Error if empty. <br/> <br/> type: New type for the credential. The supported values are: * aws_access_key * aws_iam_role * azure_ad * azure_oauth * azure_sas * azure_storage_key * docker * gcs_service_account_id * gcs_service_account_keys * hdfs * kafka <br/> identity: New user for the credential <br/> secret: New password for the credential <br/> schema_name: Updates the schema name. If *schema_name* doesn't exist, an error will be thrown. If *schema_name* is empty, then the user's default schema will be used. |
| options | map of string to strings | Optional parameters. |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| credential_name | string | Value of input parameter *credential_name*. |
| info | map of string to strings | Additional information. |
