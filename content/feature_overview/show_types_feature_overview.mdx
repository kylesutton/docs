<a id="show-types-feature-overview"></a>


Retrieves information for the specified data type ID or type label. For all data types that match the input criteria, the database returns the type ID, the type schema, the label (if available), and the type's column properties.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| type_id | string | Type Id returned in response to a call to [Create Type](/content/feature_overview/create_type_feature_overview#create-type-feature-overview) . |
| label | string | Option string that was supplied by user in a call to [Create Type](/content/feature_overview/create_type_feature_overview#create-type-feature-overview) . |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). <br/> <br/> no_join_types: When set to 'true', no join types will be included. The default value is `false`. The supported values are: * true * false |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| type_ids | array of strings |  |
| type_schemas | array of strings |  |
| labels | array of strings |  |
| properties | array of maps of string to arrays of strings |  |
| info | map of string to strings | Additional information. |
