<a id="drop-catalog-feature-overview"></a>


Drops an existing catalog.  Any external
tables that depend on the catalog must be dropped before it can be dropped.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Name of the catalog to be dropped. Must be an existing catalog. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Value of input parameter *name*. |
| info | map of string to strings | Additional information. |
