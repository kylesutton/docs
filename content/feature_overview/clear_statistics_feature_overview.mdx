<a id="clear-statistics-feature-overview"></a>


Clears statistics (cardinality, mean value, etc.) for a column in a specified table.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| table_name | string | Name of a table, in [schema_name.]table_name format, using standard [name resolution rules](../../concepts/tables/#table-name-resolution). Must be an existing table.  The default value is ''. |
| column_name | string | Name of the column in input parameter *table_name* for which to clear statistics. The column must be from an existing table. An empty string clears statistics for all columns in the table.  The default value is ''. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| table_name | string | Value of input parameter *table_name*. |
| column_name | string | Value of input parameter *column_name*. |
| info | map of string to strings | Additional information. |
