<a id="clear-tablemonitor-feature-overview"></a>


Deactivates a table monitor previously created with [Create Table Monitor](/content/feature_overview/create_tablemonitor_feature_overview#create-tablemonitor-feature-overview).


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| topic_id | string | The topic ID returned by [Create Table Monitor](/content/feature_overview/create_tablemonitor_feature_overview#create-tablemonitor-feature-overview) . |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). <br/> <br/> keep_autogenerated_sink: If *true*, the auto-generated [datasink](../../concepts/data_sinks/) associated with this monitor, if there is one, will be retained for further use. If *false*, then the auto-generated sink will be dropped if there are no other monitors referencing it. The default value is `false`. The supported values are: * true * false <br/> <br/> clear_all_references: If *true*, all references that share the same input parameter *topic_id* will be cleared. The default value is `false`.  The supported values are: * true * false |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| topic_id | string | Value of input parameter *topic_id*. |
| info | map of string to strings | Additional information. |
