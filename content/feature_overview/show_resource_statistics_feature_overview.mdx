<a id="show-resource-statistics-feature-overview"></a>


Requests various statistics for storage/memory tiers and resource groups.
Returns statistics on a per-rank basis.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| statistics_map | map of string to strings | Map of resource statistics |
| info | map of string to strings | Additional information. |
