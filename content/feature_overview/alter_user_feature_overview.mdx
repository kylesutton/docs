<a id="alter-user-feature-overview"></a>


Alters a user.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Name of the user to be altered. Must be an existing user. |
| action | string | Modification operation to be applied to the user. <br/> <br/> set_activated: Is the user allowed to login. +-----------+---------------------+ \|Supported \|Description \| \|Values \| \| +===========+=====================+ \|true \|User may login \| +-----------+---------------------+ \|false \|User may not login \| +-----------+---------------------+ <br/> <br/> set_comment: Sets the comment for an internal user. <br/> <br/> set_default_schema: Set the default_schema for an internal user. An empty string means the user will have no default schema. <br/> <br/> set_password: Sets the password of the user. The user must be an internal user. <br/> <br/> set_resource_group: Sets the resource group for an internal user. The resource group must exist, otherwise, an empty string assigns the user to the default resource group. |
| value | string | The value of the modification, depending on input parameter *action*. |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| name | string | Value of input parameter *name*. |
| info | map of string to strings | Additional information. |
