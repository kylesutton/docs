```
URL: http://<db.host>:<db.port>/show/resource/statistics
```
Requests various statistics for storage/memory tiers and resource groups.
Returns statistics on a per-rank basis.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| options | map of string to strings | Optional parameters.  The default value is an empty map ( \{\} ). |


## Output Parameter Description


The GPUdb server embeds the endpoint response inside a standard response structure which contains status information and the actual response to the query.  Here is a description of the various fields of the wrapper:

| Name | Type | Description |
| --- | --- | --- |
| status | String | 'OK' or 'ERROR' |
| message | String | Empty if success or an error message |
| data_type | String | 'show_resource_statistics_response' or 'none' in case of an error |
| data | String | Empty string |
| data_str | JSON or String | This embedded JSON represents the result of the /show/resource/statistics endpoint: <br/> <br/> statistics_map: [map of string to strings] Map of resource statistics <br/> <br/> info: [map of string to strings] Additional information. <br/> <br/> Empty string in case of an error. |
